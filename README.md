# 📊 Live Indian Stock Market Trade Signal Predictor

A Machine Learning based web application that predicts the next trading signal (BUY / SELL / HOLD) for selected Indian stocks using an LSTM Deep Learning model.

The application fetches real-time stock market data and provides an educational decision support system for stock market analysis.

---

## 🚀 Features

- 📈 Live Indian stock data using Yahoo Finance
- 🤖 LSTM Deep Learning model for prediction
- 📊 Interactive stock price visualization
- 💡 BUY / SELL / HOLD trade signals
- 🌐 Web application using Streamlit
- 📉 Model performance evaluation using RMSE

---

## 🛠 Technologies Used

- Python
- Streamlit
- TensorFlow / Keras
- NumPy
- Scikit-learn
- Matplotlib
- Yahoo Finance API
